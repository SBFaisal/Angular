ng-container

The <ng-container> is a logical container that can be used to group elements without adding extra nodes to the DOM. It is particularly useful when you want to apply structural directives (like *ngIf, *ngFor) without introducing additional HTML elements.

In this example, we use <ng-container> to conditionally render a list of items. If the list is empty, a message is displayed instead.
